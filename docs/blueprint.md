# **App Name**: Aoyama Elevator Reimagined

## Core Features:

- Homepage with Elevator Showcase: Visually present Aoyama Elevator's key projects and services with a prominent image slider.
- Services Overview: Detail the range of elevator services offered, such as installation, maintenance, and modernization.
- Project Portfolio: Display past projects with high-quality images and descriptions, focusing on various building types and elevator solutions. Enable basic filtering and sorting by type of project or location.
- About Us: Present the company's history, mission, and team.
- Contact Form: Enable potential clients to send inquiries.
- Responsive Design: Ensure the website is fully responsive and adapts to different screen sizes for optimal viewing on all devices.

## Style Guidelines:

- Primary color: Deep navy blue (#1A237E) for a sense of stability and trust.
- Background color: Very light grey (#F5F5F5) for a clean and modern feel.
- Accent color: Muted gold (#A98D5A) to highlight key elements and calls to action.
- Body and headline font: 'Inter', a versatile sans-serif suitable for both headlines and body text.
- Simple, geometric icons to represent elevator features and services.
- Clean, modern layout with a focus on showcasing high-quality project images.
- Subtle transitions and animations to create a smooth user experience.
